\item
اكتب برنامجًا يستقبل 25 عددًا صحيحًا ويطبع عدد الأعداد التي كانت أصغر من المتوسط الجزئي حتى تلك اللحظة
(أي: في اللحظة i، نحسب متوسط أوّل i أعداد، ونفحص هل العدد i أصغر من هذا المتوسط).
\ifdetailed
\begin{boxExample}
إذا كانت أول 5 أرقام مدخلة هي: $12, 8, 20, 4, 6$ فإنّ البرنامج يحسب كالتالي:
\begin{boxCode}
\begin{enumerate}[label=(\arabic*)]
    \item عندما نقرأ الرقم 12، المتوسط الجزئي هو 12. \\
    12 ليس أصغر من 12 $\leftarrow$ لا نعدّ
    \item عندما نقرأ الرقم 8، المتوسط الجزئي هو  $\frac{12 + 8}{2} = 10$. \\
    8 أصغر من 10 $\leftarrow$ نعدّ (العداد الآن 1)
    \item عندما نقرأ الرقم 20، المتوسط الجزئي هو  $\frac{12 + 8 + 20}{3} = 13.33$. \\
    20 ليس أصغر من 13.33 $\leftarrow$ لا نعدّ (العداد الآن 1)
    \item عندما نقرأ الرقم 4، المتوسط الجزئي هو  $\frac{12 + 8 + 20 + 4}{4} = 11$. \\
    4 أصغر من 11 $\leftarrow$ نعدّ (العداد الآن 2)
    \item عندما نقرأ الرقم 6، المتوسط الجزئي هو $\frac{12 + 8 + 20 + 4 + 6}{5} = 10$. \\
    6 أصغر من 10 $\leftarrow$ نعدّ (العداد الآن 3)
\end{enumerate}
\end{boxCode}
\end{boxExample}
\fi
\ifwithsols
\begin{boxSolution}
\begin{english}
\begin{minted}{csharp}
int sum = 0;
int cnt = 0;
double avg;

for(int i=1; i<=25; i++)
{
    int x = int.Parse(Console.ReadLine());
    sum += x;
    avg = sum / (double)i;
    if (x < avg)
        cnt++;
}
Console.WriteLine(cnt);
\end{minted}
\end{english}
\end{boxSolution}
\clearpage
\fi

\item اكتب عملية تتلقى عددًا صحيحًا $n$، وتحسب قيمة التعبير التالي: \\

public static int AlternatingSum(int n)
{
    int sum = 0;

    {
        if (i % 2 == 0)
            sum -= i;
        else
            sum += i;
    }

}

\item اكتب برنامجًا يستقبل من المستخدم 20 زوجًا من الأعداد (أي ما مجمله 40 عددًا.) \\
لكل زوج من الأعداد، يحسب البرنامج معدّل العددين ويطبعه. \\
وفي النهاية يطبع العددين اللذين كان لهما أكبر معدّل.
\ifdetailed
\begin{boxExample}
إذا كان البرنامج سيستقبل 5 أزواج فقط، وهذه الأزواج هي: $(2,4), (5,7), (1,9), (6,8), (3,3)$ فإنّ البرنامج يعمل كما يلي:
\begin{boxCode}
    \begin{enumerate}[label=(\arabic*)]
        \item يستقبل الرقمين 2 و 4، المعدّل هو $\frac{2+4}{2} = 3$. يطبع 3.
        \item يستقبل الرقمين 5 و 7، المعدّل هو $\frac{5+7}{2} = 6$. يطبع 6.
        \item يستقبل الرقمين 1 و 9، المعدّل هو $\frac{1+9}{2} = 5$. يطبع 5.
        \item يستقبل الرقمين 6 و 8، المعدّل هو $\frac{6+8}{2} = 7$. يطبع 7.
        \item يستقبل الرقمين 3 و 3، المعدّل هو $\frac{3+3}{2} = 3$. يطبع 3.
    \end{enumerate}
    في النهاية، يطبع العددين اللذين كان لهما أكبر معدّل: 6 و 8 (لأنّ معدلهما هو 7 وهو الأكبر بين المعدّلات).
\end{boxCode}
\end{boxExample}
\fi
\ifwithsols
\begin{boxSolution}
\begin{english}
\begin{minted}{csharp}
double maxAvg = double.MinValue;
int num1 = 0, num2 = 0;
for (int i = 0; i < 20; i++)
{
    int a = int.Parse(Console.ReadLine());
    int b = int.Parse(Console.ReadLine());
    double avg = (a + b) / 2.0;
    Console.WriteLine(avg);
    if (avg > maxAvg)
    {
        maxAvg = avg;
        num1 = a;
        num2 = b;
    }
}
Console.WriteLine(num1);
Console.WriteLine(num2);
\end{minted}
\end{english}
\end{boxSolution}
\clearpage
\fi

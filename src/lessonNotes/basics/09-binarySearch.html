فكرة من الحياة اليومية

تخيّل أنك تبحث عن كلمة في قاموس يحتوي على 1000 صفحة. هل تبدأ من الصفحة الأولى وتقرأ صفحة صفحة؟

بالطبع لا! أنت تفتح القاموس من المنتصف، وتقرر:

  • إذا الكلمة التي تبحث عنها تأتي بعد الصفحة المفتوحة → تتجاهل النصف الأيسر كله.
  • إذا تأتي قبلها → تتجاهل النصف الأيمن كله.
  • تكرر هذا حتى تجد الكلمة.

هذا بالضبط هو البحث الثنائي!
في كل خطوة تحذف نصف العناصر المتبقية. فبدلاً من فحص 1000 عنصر واحداً واحداً، تحتاج فقط إلى 10 خطوات (لأن 2¹⁰ = 1024).

شرط أساسي: هذه الفكرة تعمل فقط لأن القاموس مرتّب أبجدياً. بدون ترتيب، لا يمكن أن نقرر أي نصف نتجاهله!

خطوات الخوارزمية

البحث الثنائي يعمل على مصفوفة مرتبة تصاعدياً كالتالي:

  1. نُعرّف ثلاثة متغيرات: lo = 0، hi = arr.Length - 1، mid.
  2. طالما lo <= hi، نحسب الوسط: mid = (lo + hi) / 2
  3. نقارن arr[mid] مع القيمة المطلوبة target:
    • إذا arr[mid] == target → وجدنا! نُعيد mid.
    • إذا arr[mid] < target → نرفع الحد الأدنى: lo = mid + 1
    • إذا arr[mid] > target → نخفض الحد الأعلى: hi = mid - 1
  4. إذا خرجنا من الحلقة بدون إيجاد → نُعيد ‎-1.
اللون الأخضر = lo  |  اللون الأحمر = hi  |  اللون الأصفر = mid  |  رمادي باهت = نطاق محذوف

كود C# الكامل

public static int BinarySearch(int[] arr, int target) { int lo = 0; int hi = arr.Length - 1; while (lo <= hi) { int mid = (lo + hi) / 2; if (arr[mid] == target) return mid; // وجدنا → نُعيد الفهرس else if (arr[mid] < target) lo = mid + 1; // المطلوب في النصف الأيمن else hi = mid - 1; // المطلوب في النصف الأيسر } return -1; // لم نجده }

ماذا تُعيد العملية؟ إذا وُجد العنصر تُعيد فهرسه (index). إذا لم يُوجد تُعيد ‎-1 — وهي الاتفاقية المعيارية للدلالة على "غير موجود".

العملية تُعيد -1

التتبع:

  • الخطوة 1: lo=0، hi=6 → mid=3 → arr[3]=12 > 10 → hi = 2
  • الخطوة 2: lo=0، hi=2 → mid=1 → arr[1]=5 < 10 → lo = 2

شرط lo <= hi هو "صمام الأمان" الذي يوقف الحلقة عندما يضيق النطاق ونُدرك أن العنصر غير موجود.

نعكس الشرط: نغير < إلى >

else if (arr[mid] > target) lo = mid + 1; // المطلوب في النصف الأيمن (الأصغر) else hi = mid - 1; // المطلوب في النصف الأيسر (الأكبر)

التفسير:

في الترتيب التنازلي: إذا arr[mid] أكبر من target → نتجه يميناً أيضاً (لأن اليمين أصغر الآن).
